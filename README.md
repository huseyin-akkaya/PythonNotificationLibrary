# Python Masaüstü Bildirim Uygulaması

Python ve `notifypy` kütüphanesi kullanılarak geliştirilmiş, özelleştirilebilir bir masaüstü bildirim (desktop notification) betiği.

## Özellikler
* İşletim sisteminin yerel bildirim merkezine entegre mesaj gönderimi.
* Masaüstüne özel başlık ve içerik metni yansıtma.
* Harici bir görsel (`icon.png`) kullanarak bildirim ikonunu özelleştirme.

## Kullanılan Teknolojiler
* Python
* `notify-py` Kütüphanesi

## Nasıl Çalıştırılır?

Terminal üzerinden projenin bulunduğu dizine gidin, gerekli kütüphaneyi kurun ve betiği çalıştırın:

```bash
pip install notify-py
python main.py
